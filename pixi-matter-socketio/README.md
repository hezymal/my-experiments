# My Experiments: PixiJS and MatterJS

Эксперимент с библиотеками `PixiJS` (библиотека для работы с 2D графикой в браузере) и `MatterJS` (библиотека 2D физики).

## Скрипты

```bash
# запуск приложения
yarn start
```
