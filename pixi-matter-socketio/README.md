# pixi-matter-socketio

Эксперимент с библиотеками `pixi.js` (библиотека для работы с 2D графикой в браузере), `matter.js` (библиотека 2D физики) и `socket.io` (библиотека для обмена сообщениями).

## Скрипты

```bash
# запуск приложения
yarn start
```
