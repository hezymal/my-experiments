# three-html2canvas

Эксперимент с библиотеками `three` (библиотека для работы с 3D графикой в браузере) и `html2canvas` (библиотека умеющая делать скриншоты). Пример эффекта перехода между формами.

## Скрипты

```bash
# Запуск приложения
yarn start
```
